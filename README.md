## ◈ How GitHub Sync Works

```
┌─────────────────────────────────────────────────────────┐
│                     SYNC PIPELINE                       │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   [NeetCode Editor]                                     │
│         │                                               │
│         ▼  Submit Solution                              │
│   [NeetCode Backend]                                    │
│         │                                               │
│         ▼  Auto-commit (if enabled)                     │
│   [GitHub Repository]  ◄──── Bulk Sync (past runs)     │
│         │                                               │
│         ▼  Manual override                              │
│   [Submission History Panel]                            │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

| Step | Action |
|------|--------|
| **01 · Connect** | Link your GitHub at [neetcode.io/profile/github](https://neetcode.io/profile/github) |
| **02 · Auto-commit** | Every accepted submission is pushed automatically |
| **03 · Bulk Sync** | Push all past solutions at once from the settings page |
| **04 · Manual Sync** | Fine-tune individual submissions from the history panel |
